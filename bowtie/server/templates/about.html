About Bowtie

Sensor Data Gathering

Bowtie is a sensor integration server using HTML5 and was written in Python and Javascript. This means sensor data can be collected from most devices and pushed to a server. The sensor data is stored in a unique folder defined by the CPU Id and inside the folder are JSON files that are defined by the Node Id. This makes it easy in any programming language on almost any operating system to gather the sensor data.
